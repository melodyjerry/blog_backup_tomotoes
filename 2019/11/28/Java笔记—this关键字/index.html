this 关键字

三个形式上的用途，但本质上就两个：

  • 调用重载的其他构造方法：放置在构造方法体首行的this()或this(实参列表)
  • 作为当前对象，引用成员（数据、方法等）：区分同名变量等
  • 作为当前对象，方法返回当前对象的引用，形成链式调用
